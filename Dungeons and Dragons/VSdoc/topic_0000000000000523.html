ChatClient.ConnectAndSetStatus Method

Namespace: Photon.Chat
Assembly: Assembly-CSharp (in Assembly-CSharp.dll)
Connects this client to the Photon Chat Cloud service, which will also authenticate the user (and set a UserId). This also sets an online status once connected. By default it will set user status to ChatUserStatus.Online. See SetOnlineStatus for more information.

Get your Photon Chat AppId from the Dashboard.

appVersion

Any version string you make up. Used to separate users and variants of your clients, which might be incompatible.

authValues
AuthenticationValues

Values for authentication. You can leave this null, if you set a UserId before. If you set authValues, they will override any UserId set before.

status
int

User status to set when connected. Predefined states are in class ChatUserStatus. Other values can be used at will.
