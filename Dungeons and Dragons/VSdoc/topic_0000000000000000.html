PhotonNetworkingMessage Enumeration

Namespace: <default>

Remarks
 
Much like "Update()" in Unity, PUN will call methods in specific situations. Often, these methods are triggered when network operations complete (example: when joining a room). All those methods are defined and described in this enum and implemented by PunBehaviour (which makes it easy to implement them as override). Each entry is the name of such a method and the description tells you when it gets used by PUN. Make sure to read the remarks per entry as some methods have optional parameters.
