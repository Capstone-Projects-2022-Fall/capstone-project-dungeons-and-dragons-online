countOfPlayersOnMaster The count of players currently looking for a room (available on MasterServer in 5sec intervals).
countOfRooms The count of rooms currently in use (available on MasterServer in 5sec intervals).

masterClient The Master Client of the current room or null (outside of rooms).

otherPlayers The list of players in the current room, excluding the local player.
player The local PhotonPlayer. Always available and represents this player. CustomProperties can be set before entering a room and will be synced as well.
playerList The list of players in the current room, including the local player.
playerName Set to synchronize the player's nickname with everyone in the room(s) you enter. This sets PhotonNetwork.player.NickName.
room Get the room we're currently in. Null if we aren't in any room.
sendRate Defines how many times per second PhotonNetwork should send a package. If you change this, do not forget to also change 'sendRateOnSerialize'.
sendRateOnSerialize Defines how many times per second OnPhotonSerialize should be called on PhotonViews.
