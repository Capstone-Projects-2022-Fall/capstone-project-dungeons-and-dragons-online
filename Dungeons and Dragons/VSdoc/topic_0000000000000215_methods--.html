PhotonNetwork Methods

The following tables list the members exposed by the PhotonNetwork type.

Methods
 
Name Description
AllocateSceneViewID() Enables the Master Client to allocate a viewID that is valid for scene objects.
AllocateViewID() Allocates a viewID that's valid for the current/local player.
CacheSendMonoMessageTargets(Type) Populates SendMonoMessageTargets with currently existing GameObjects that have a Component of type.
CallEvent(byte, object, int)
CloseConnection(PhotonPlayer) Request a client to disconnect (KICK). Only the master client can do this
ConnectToBestCloudServer(string) Connect to the Photon Cloud region with the lowest ping (on platforms that support Unity's Ping).
ConnectToMaster(string, int, string, string) Connect to a Photon Master Server by address, port, appID and game(client) version.
ConnectToRegion(CloudRegionCode, string, string) Connects to the Photon Cloud region and cluster of choice.
ConnectUsingSettings(string) Connect to Photon as configured in the editor (saved in PhotonServerSettings file).
CreateRoom(string) Creates a room with given name but fails if this room(name) is existing already. Creates random name for roomName null.
CreateRoom(string, RoomOptions, TypedLobby) Creates a room but fails if this room is existing already. Can only be called on Master Server.
CreateRoom(string, RoomOptions, TypedLobby, string[]) Creates a room but fails if this room is existing already. Can only be called on Master Server.
CreateSettings()
Destroy(GameObject) Network-Destroy the GameObject, unless it is static or not under this client's control.
Destroy(PhotonView) Network-Destroy the GameObject associated with the PhotonView, unless the PhotonView is static or not under this client's control.
DestroyAll() Network-Destroy all GameObjects, PhotonViews and their RPCs in the room. Removes anything buffered from the server. Can only be called by Master Client (for anyone).
DestroyPlayerObjects(PhotonPlayer) Network-Destroy all GameObjects, PhotonViews and their RPCs of targetPlayer. Can only be called on local player (for "self") or Master Client (for anyone).
DestroyPlayerObjects(int) Network-Destroy all GameObjects, PhotonViews and their RPCs of this player (by ID). Can only be called on local player (for "self") or Master Client (for anyone).
Disconnect() Makes this client disconnect from the photon server, a process that leaves any room and calls OnDisconnectedFromPhoton on completion.
FetchServerTimestamp() Refreshes the server timestamp (async operation, takes a roundtrip).
FindAssetPath(string) Finds the asset path base on its name or search query: https://docs.unity3d.com/ScriptReference/AssetDatabase.FindAssets.html
FindFriends(string[], FindFriendsOptions) Requests the rooms and online status for a list of friends and saves the result in PhotonNetwork.Friends.
FindGameObjectsWithComponent(Type) Finds the GameObjects with Components of a specific type (using FindObjectsOfType).
FindPunAssetFolder() Finds the pun asset folder. Something like Assets/Photon Unity Networking/Resources/
GetCustomRoomList(TypedLobby, string) Fetches a custom list of games from the server, matching a SQL-like "where" clause, then triggers OnReceivedRoomListUpdate callback.
GetParent(string, string) Gets the parent directory of a path. Recursive Function, will return null if parentName not found
GetPing() The current roundtrip time to the photon server.
GetRoomList() Gets currently cached rooms of the last rooms list sent by the server as RoomInfo array. This list is either available and updated automatically and periodically while in a lobby (check insideLobby) or received as a response to PhotonNetwork.GetCustomRoomList().
InitializeSecurity() Obsolete. Used for compatibility with Unity networking only. Encryption is automatically initialized while connecting.
Instantiate(string, Vector3, Quaternion, byte) Instantiate a prefab over the network. This prefab needs to be located in the root of a "Resources" folder.
Instantiate(string, Vector3, Quaternion, byte, object[]) Instantiate a prefab over the network. This prefab needs to be located in the root of a "Resources" folder.
InstantiateSceneObject(string, Vector3, Quaternion, byte, object[]) Instantiate a scene-owned prefab over the network. The PhotonViews will be controllable by the MasterClient. This prefab needs to be located in the root of a "Resources" folder.
InternalCleanPhotonMonoFromSceneIfStuck() Internally used by Editor scripts, called on Hierarchy change (includes scene save) to remove surplus hidden PhotonHandlers.
JoinLobby() On MasterServer this joins the default lobby which list rooms currently in use.
JoinLobby(TypedLobby) On a Master Server you can join a lobby to get lists of available rooms.
JoinOrCreateRoom(string, RoomOptions, TypedLobby) Lets you either join a named room or create it on the fly - you don't have to know if someone created the room already.
JoinOrCreateRoom(string, RoomOptions, TypedLobby, string[]) Lets you either join a named room or create it on the fly - you don't have to know if someone created the room already.
JoinRandomRoom() Joins any available room of the currently used lobby and fails if none is available.
JoinRandomRoom(Hashtable, byte) Attempts to join an open room with fitting, custom properties but fails if none is currently available.
JoinRandomRoom(Hashtable, byte, MatchmakingMode, TypedLobby, string, string[]) Attempts to join an open room with fitting, custom properties but fails if none is currently available.
JoinRoom(string) Join room by roomname and on success calls OnJoinedRoom(). This is not affected by lobbies.
JoinRoom(string, string[]) Join room by roomname and on success calls OnJoinedRoom(). This is not affected by lobbies.
LeaveLobby() Leave a lobby to stop getting updates about available rooms.
LeaveRoom(bool) Leave the current room and return to the Master Server where you can join or create rooms (see remarks).
LoadLevel(int) Wraps loading a level to pause the network message-queue. Optionally syncs the loaded level in a room.
LoadLevel(string) Wraps loading a level to pause the network message-queue. Optionally syncs the loaded level in a room.
LoadLevelAsync(int) Wraps single asynchronous loading of a level to pause the network message-queue. Optionally syncs the loaded level in a room.
LoadLevelAsync(string) Wraps single asynchronous loading of a level to pause the network message-queue. Optionally syncs the loaded level in a room.
NetworkStatisticsReset() Resets the traffic stats and re-enables them.
NetworkStatisticsToString() Only available when NetworkStatisticsEnabled was used to gather some stats.
OverrideBestCloudServer(CloudRegionCode) Overwrites the region that is used for ConnectToBestCloudServer(string gameVersion).
RaiseEvent(byte, object, bool, RaiseEventOptions) Sends fully customizable events in a room. Events consist of at least an EventCode (0..199) and can have content.
ReJoinRoom(string) Can be used to return to a room after a disconnect and reconnect.
Reconnect() Can be used to reconnect to the master server after a disconnect.
ReconnectAndRejoin() When the client lost connection during gameplay, this method attempts to reconnect and rejoin the room.
RefreshCloudServerRating() Pings all cloud servers again to find the one with best ping (currently).
RemovePlayerCustomProperties(string[]) Locally removes Custom Properties of "this" player. Important: This does not synchronize the change! Useful when you switch rooms.
RemoveRPCs(PhotonPlayer) Remove all buffered RPCs from server that were sent by targetPlayer. Can only be called on local player (for "self") or Master Client (for anyone).
RemoveRPCs(PhotonView) Remove all buffered RPCs from server that were sent via targetPhotonView. The Master Client and the owner of the targetPhotonView may call this.
RemoveRPCsInGroup(int) Remove all buffered RPCs from server that were sent in the targetGroup, if this is the Master Client or if this controls the individual PhotonView.
SendOutgoingCommands() Can be used to immediately send the RPCs and Instantiates just called, so they are on their way to the other players.
SetInterestGroups(byte, bool) Enable/disable receiving events from a given Interest Group.
SetInterestGroups(byte[], byte[]) Enable/disable receiving on given Interest Groups (applied to PhotonViews).
SetLevelPrefix(short) Sets level prefix for PhotonViews instantiated later on. Don't set it if you need only one!
SetMasterClient(PhotonPlayer) Asks the server to assign another player as Master Client of your current room.
SetPlayerCustomProperties(Hashtable) Sets this (local) player's properties and synchronizes them to the other players (don't modify them directly).
SetReceivingEnabled(int, bool) Obsolete.
SetReceivingEnabled(int[], int[]) Obsolete.
SetSendingEnabled(int, bool) Obsolete.
SetSendingEnabled(int[], int[]) Obsolete.
SetSendingEnabled(byte, bool) Enable/disable sending on given group (applied to PhotonViews)
SetSendingEnabled(byte[], byte[]) Enable/disable sending on given groups (applied to PhotonViews)
SwitchToProtocol(ConnectionProtocol) While offline, the network protocol can be switched (which affects the ports you can use to connect).
UnAllocateViewID(int) Unregister a viewID (of manually instantiated and destroyed networked objects).
WebRpc(string, object) This operation makes Photon call your custom web-service by name (path) with the given parameters.
